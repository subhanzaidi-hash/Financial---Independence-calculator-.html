# Financial Independence Calculator

A modern, interactive Financial Independence (FI) Calculator that helps users estimate the age at which they can achieve financial independence based on their savings, income, expenses, investment returns, inflation, and withdrawal rate.

## Features

- Supports 14 global currencies:
  - USD
  - EUR
  - GBP
  - AED
  - SAR
  - PKR
  - INR
  - JPY
  - CNY
  - CAD
  - AUD
  - CHF
  - TRY
  - SGD

- Real-time calculation of:
  - Financial Independence Age
  - FI Number Required
  - Savings Rate
  - Years to Financial Independence
  - Projected Portfolio Value

- Scenario Analysis:
  - Conservative Scenario
  - Base Case Scenario
  - Optimistic Scenario

- Interactive Portfolio Growth Chart

- Mobile-Friendly Responsive Design

- Light and Dark Mode Support
